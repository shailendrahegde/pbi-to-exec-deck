# Push to GitHub - Quick Commands

Repository name: **pbi-to-exec-deck**

## Quick Push (Copy & Paste)

```bash
cd C:\Users\shahegde\claudex\github-repo

# Initialize git
git init

# Add all files
git add .

# Initial commit
git commit -m "Initial commit: Power BI to Executive Deck v1.4"

# Add remote (replace [your-username] with your GitHub username)
git remote add origin https://github.com/[your-username]/pbi-to-exec-deck.git

# Push to main branch
git branch -M main
git push -u origin main
```

## Alternative: Using GitHub CLI

```bash
cd C:\Users\shahegde\claudex\github-repo

gh repo create pbi-to-exec-deck --public --source=. --push --description "Transform Power BI dashboards into executive-ready presentations using Claude Code"
```
